# To-Do List Flask App

## 🔹 Description
This is my first web application built using Python Flask.
It allows users to add and delete tasks easily.

## 🔹 Features
- Add tasks
- Delete tasks
- View tasks

## 🔹 Technologies Used
- Python
- Flask
- HTML
- CSS

## 🔹 How to Run
1. Clone the repository
2. Install Flask (pip install flask)
3. Run app.py
4. Open http://127.0.0.1:5000/
